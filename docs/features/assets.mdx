# Assets

The Assets page can be used to track non-account-based assets, such as property, vehicles, or other valuables.

## Creating

To create a new asset, click on the plus icon in the top right of the `Assets` page. Fill out the form with a unique name for the asset.

## Categorizing

Assets can be assigned a type to help group them for reporting and insights, similar to accounts.
There are a set of built-in types that should cover most use cases, but you can also create custom types from the asset settings page.

## Editable Fields

To edit an asset, click the edit button (pencil icon) next to the asset name. There are a few fields you can modify:

| Field          | Description                                                  |
| -------------- | ------------------------------------------------------------ |
| Name           | Asset display name. Must be unique.                          |
| Purchase Date  | The date you acquired the asset.                             |
| Purchase Price | The amount paid when you acquired the asset.                 |
| Sell Date      | The date you sold the asset (if applicable).                 |
| Sell Price     | The amount received when you sold the asset (if applicable). |

## Tracking value

Value information can be found on the asset detail page. Click on an asset to view its details, including a history of value entries.

### Adding value entries

To add a value entry, navigate to the `Add Value` section, where you can input the date and value of the asset at that time.

### Managing value entries

All value entries for the asset will be listed in the `Value History` section. You can edit or delete individual entries using the corresponding buttons next to each entry.
