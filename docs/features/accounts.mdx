# Accounts

Accounts represent your financial accounts, such as bank accounts, credit cards, cash, etc. You can create and manage accounts on the `Accounts` page.

## Account Structure

Accounts are grouped using the following structure:

```
- Institution
|-- Account
|-- Account
- Institution
|-- Account
...
```

Institutions represent the financial institution that an account belongs to (e.g., Chase, Wells Fargo, etc.). A financial institution can have multiple accounts, but an account can only belong to one institution.

## Creating Accounts

To create a new account, click on the plus icon on the top right of the `Accounts` page. Fill out the form with the account details:

- **Account Name**: The name of the account (e.g., "Checking", "Savings", "Credit Card").
- **Institution**: Specify either an existing institution or create a new one by typing the name and pressing Enter.

## Reordering

You can change the order in which accounts are displayed both on the `Accounts` page and in the `Accounts` card on the dashboard. To reorder accounts, click the `Reorder` button in the top right, then drag and drop accounts to your desired order using the provided buttons. Click `Save` when finished.

:::tip
You can reorder institutions as well by dragging and dropping them in the same way.
:::

## Categorizing

Accounts can be assigned a type to help group them for reporting and insights.
There are a set of built-in types that should cover most use cases, but you can also create custom types from the accounts settings page.

### Account Type Classification

Account types can be classified as either asset or liability types.
This classification is used to determine how the account balance contributes to your net worth and other financial metrics.

## Editable Fields

To edit an account or institution, click the edit button (pencil icon) next to the account or institution name. Make your changes in the form that appears.

### Institutions

| Field | Description                          |
| ----- | ------------------------------------ |
| Name  | Update the institution display name. |

### Accounts

| Field             | Description                                                                                                                                                                                        |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name              | Account display name.                                                                                                                                                                              |
| Interest Rate     | Optional rate for interest-bearing accounts (for example, savings or loans). Used to improve goal calculations.                                                                                    |
| Account Type      | Used for grouping and reporting. Helps distinguish asset accounts (for example, checking/savings) from liability accounts (for example, credit cards/loans).                                       |
| Hide Account      | Hides the account from dashboard summaries and reports while keeping historical data.                                                                                                              |
| Hide Transactions | Hides this account's transactions from the transaction list and spending metrics to avoid double counting in common scenarios (for example, loan accounts).                                        |
| Delete Account    | Deletes the account, with an option to also delete associated transactions. Deleted accounts can be restored, but deleted transactions must be restored individually from the `Transactions` page. |

## Account details

Clicking on an account will bring up the account details page, where you can see more details about the balance activity of the account.

### Add balance

This section will allow you to manually update the balance of the account.

### Account trends

This section shows a chart of the account balance over time.

### Recent balances

This section shows a list of recent balance updates for the account. Balance updates are performed automatically when syncing with SimpleFIN or importing transactions via CSV, but can also be done manually using the `Add balance` section above.
