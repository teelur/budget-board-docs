## New Features

### Date Format Changes

I've updated the Date Format setting to now allow for setting the date separator. You can now customize using any non-alphanumeric character (e.g., `/`, `-`,`.`, etc.).

### Trends Graph Updates

- Fixed a bug with the Net Worth graph adding duplicate months when using the `Last N Months` buttons
- Added the `Last N Months` buttons to all graphs

### Other changes

- Bunch of translation updates from Weblate
- Fixed a CVE with a dependency
