# Translations

## Weblate

We use [Weblate](https://weblate.org/en/) to manage translations for Budget Board. If you'd like to contribute translations, please visit our Weblate project [here](https://hosted.weblate.org/engage/budget-board/).

:::tip
You can submit a request for a new language [here](https://hosted.weblate.org/new-lang/budget-board/).
:::
