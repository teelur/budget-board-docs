# Overview

## Summary

BudgetBoard is an open-source personal finance application designed to help users track their monthly spending and work towards financial goals. It offers a user-friendly interface, robust budgeting tools, and seamless integration with various financial institutions via sync providers like SimpleFIN.

### Key Features

- **Dashboard**: Provides an overview of your financial situation, including account balances, uncategorized transactions, and monthly spending trends.
- **Accounts Management**: Add, edit, and manage multiple financial accounts, including checking, savings, and credit card accounts.
- **Assets Tracking**: Track various assets, such as properties and other valuables, monitoring purchase and sale prices as well as value changes over time.
- **Transaction Management**: Add, edit, and categorize transactions.
- **Transaction CSV Import**: Allows users to import transactions in bulk using CSV files.
- **Automatic Rules**: Create rules to automatically update transactions based on criteria like description, amount, or date.
- **Budgeting Tools**: Create and manage budgets on a monthly basis, and track spending against those budgets.
- **Sync Providers**: Supports integration with financial institutions through providers like SimpleFIN for automatic transaction and account balance syncing.
- **Customizable Trends Charts**: Visualize spending trends with customizable charts that can be filtered by date range, account, and category.
- **User Authentication**: Supports local authentication with two-factor authentication (2FA), or you can bring your own authentication provider with OIDC login.
